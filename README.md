# CompetitionTimer
Web-based Timer and Score for competition.

# README[petunjuk].txt :
```
Penggunaan Timer Web App :
[ = Timer Semi Final
] = Timer Final
\ = Timer Biasa

-------[Petunjuk Timer Semi Final]--------
S = start timer
P = pause timer
R = reset timer

N = test alarm 1
M = test alarm 2
------------------------------------------
Ubah Pengaturan di config.js
```

var background_image = "url('prima1.png')"; //Lokasi background. Background's location
